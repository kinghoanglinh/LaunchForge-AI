# DealScope AI Launch Checklist

## Before Submission

- [ ] Product type is Prompt.
- [ ] Pricing is Free for v1.
- [ ] Name is `DealScope AI`.
- [ ] Category is Finance or Research.
- [ ] Description avoids guaranteed investment outcomes.
- [ ] Prompt includes financial/legal/investment advice disclaimer.
- [ ] Prompt explicitly says not to hallucinate facts.
- [ ] Prompt handles sparse input gracefully.
- [ ] Prompt produces assumptions log.
- [ ] Prompt produces GO / CONDITIONAL GO / NO-GO verdict.
- [ ] At least 3 use cases are included.
- [ ] GitHub link is attached.
- [ ] Cover image is uploaded.

## Suggested Swarms Fields

Name:

```text
DealScope AI
```

Description:

```text
DealScope AI is a structured startup due diligence prompt for founders, angel investors, accelerator mentors, and Web3 builders. Provide a startup name, pitch, or brief company description, and DealScope generates an analyst-style memo covering market intelligence, competitive moat, team and execution, business model, risk flags, assumptions, and a GO / CONDITIONAL GO / NO-GO verdict.
```

Tags:

```text
Startup, Due Diligence, VC, Investment Research, Founder Tools, SEA, Risk Analysis, Finance
```

Use cases:

1. Validate a startup idea before building.
2. Pre-screen inbound startup pitches.
3. Create accelerator mentor feedback.
4. Analyze SEA/Vietnam startup opportunities.

## After Submission

- [ ] Check Swarms quality score.
- [ ] If score is below 3.5, improve description and prompt specificity.
- [ ] Share with 5-10 trusted testers for honest review.
- [ ] Collect output examples and refine v1.1.
- [ ] Launch paid pro version only after account is eligible.
