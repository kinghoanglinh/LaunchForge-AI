# DealScope AI Example Inputs

## Example 1: Sparse SaaS Input

```text
Analyze this startup idea:

FreshOps helps small restaurants in Vietnam manage inventory, supplier ordering,
and daily food cost from one mobile dashboard.
```

## Example 2: Rich Fintech Input

```text
Company: RemitBridge
Description: Cross-border payroll and remittance platform for Filipino remote
workers employed by US startups.
Market: Philippines, US remote work market
Customer: Remote workers and small US employers
Team: Former payroll operations manager, ex-bank compliance officer, full-stack
engineer
Traction: 220 beta users, $18K monthly transaction volume, no revenue yet
Revenue model: 0.8% transaction fee plus employer dashboard subscription
Competitors: Wise, Payoneer, local remittance shops
Ask: Should this team pursue a seed round?
```

## Example 3: Web3 Builder Input

```text
Company: GuildProof
Description: On-chain credential layer for gaming guilds and contributor
communities.
Market: Web3 gaming and contributor DAOs
Team: One smart contract engineer, one community operator, one game producer
Traction: 3 pilot communities, 9,000 wallets indexed, no paid customers
Revenue model: SaaS dashboard plus protocol fee

```

## Example 4: Marketplace Input

Company: TutorLoop

Team: Former tutoring center operator and two university student community leads
